Materializer

In order to make a dynamic Variable Card static, a Materializer can be used. The dynamic Variable Card must be inserted, and a new empty Variable Card must be inserted, which will cause it to contain the current value of the dynamic Variable Card as a static value.

This can be useful in situations where you want to log the values of a certain dynamic Variable Card, or when you donâ€™t need the dynamic evaluation of a Variable Card anymore.

Variable Cards can be piped in and out the Materializer.
