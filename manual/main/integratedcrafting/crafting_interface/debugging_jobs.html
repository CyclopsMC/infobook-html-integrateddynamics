Debugging Crafting Jobs

While setting up autocrafting, you may run into situations where you crafting job does not work properly due to misconfiguration.

You can debug these cases using crafting-related aspects from the Network Reader, such as getting the list of all available recipes in the network, getting all running crafting jobs, and getting all missing crafting job ingredients. Visualizing these lists in Display Panels can help you pinpoint the problem you are experiencing.

Note that if you have Integrated Terminals installed, you can use the Crafting Job Terminal to visualize all running crafting jobs, visualize any missing ingredients, or cancel the jobs.
