Display

A useful monitoring or debugging tool that simply shows the value of a Variable Card. Displays can be rotated by right-clicking with a Wrench.

Inside the gui, a Copy button is available to copy the currently displayed value to your clipboard so that it can be used outside of the game. You can also just use the default copy hotkey of your OS to copy this value.
