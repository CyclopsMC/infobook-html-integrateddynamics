모드로 아이템 저장소 필터링

필터링 아이템 인터페이스를 사용하여 상자를 네트워크에 연결하고 IntegratedDynamics 모드의 아이템만 전달하는 조건자 기반 필터를 구성합니다. 이를 위해 아이템을 입력으로 사용하고 Boolean을 출력으로 반환하는 새 연산자를 생성합니다.

이렇게 하려면 먼저 관계 같음 연산자가 포함된 연산자 값을 만드세요. 다음으로 IntegratedDynamics 값을 포함하는 String을 작성하세요. 그런 다음 적용 연산자를 사용하여 관게 같음 연산자를 String 값에 부분적으로 적용합니다.

다음 단계로 다른 연산자 값을 생성해야 하지만 이번에는 아이템 모드 연산자를 포함합니다. 이제 파이프 연산자를 사용하여 아이템 모드 연산자를 이전 단계에서 부분적으로 적용된 연산자로 파이프할 수 있습니다.

파이프 연산자의 이 결과는 필터링 아이템 인터페이스의 술어 기반 기능에 배치할 수 있으며 우리가 의존하는 모든 변수는 변수 저장소에 배치해야 합니다. 모든 단계가 올바르게 수행되면 필터는 IntegratedDynamics 모드의 아이템만 전달하며 이는 String 상수를 변경하여 사용자 지정할 수 있습니다.
