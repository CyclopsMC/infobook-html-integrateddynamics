레시피 피드백

변수 카드을 제작 인터페이스에 삽입할 때 녹색 확인 표시 또는 적색 십자가가 표시되어 주어진 레시피가 바인딩된 기계에 유효한지 나타냅니다. 이 기호 위로 마우스를 가져가면 더 많은 정보를 얻을 수 있습니다.

이것은 예를 들어 상자에 대한 제작법을 화로에 연결된 제작 인터페이스에 삽입하는 경우와 같이 실수로 잘못된 제작법을 인터페이스에 삽입한 경우에 대한 조기 피드백 시스템입니다.

If you see a red cross, this means that the Variable Card either doesn't expose a valid recipe, or that the recipe cannot be crafted in the bound machine.

A green checkmark indicates that the Variable Card contains a valid recipe, and that the machine can (probably) handle the given recipe.
