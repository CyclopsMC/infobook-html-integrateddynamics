제작 작성기

제작 작성기을 사용하면 제작 인터페이스에 의해 노출된 레시피를 사용하여 특정 조건에 따라 자동으로 제작 작업을 시작할 수 있습니다.

다음 기능을 사용하여 주어진 레시피, 출력 아이템, 출력 유체 또는 출력 에너지에 대한 제작 작업을 시작할 수 있습니다. 기능 설정을 사용하여 이미 실행 중인 제작 작업이 새 작업을 차단해야 하는지 여부, 누락된 입력에 대한 제작 작업도 시작해야 하는지 여부, 저장 네트워크에 재료가 있는지 여부 등의 제작 채널을 구성할 수 있습니다. 새로운 제작 작업을 차단해야 합니다.

변수 카드를 삽입하여 아무것도 제작되지 말아야 함을 알리려면 실제 아이템이 없는 아이템 자료형에 대한 변수 카드와 같이 빈 값을 삽입하기만 하면 됩니다.
