Chemical Value Type

This mod adds a dedicated value type to represent references to a Mekanism chemical. For example Sodium, Oxygen, Dirty Gold Slurry, Biomass, ....

Chemicals are also supported in Recipes and Ingredients. For chemical-based recipes, a more convenient entry in the Logic Programmer is available under the name Chemical Recipe. This has support for mods such as JEI.

Chemical values are also fully supported within mods such as Integrated Scripting and Integrated REST.
