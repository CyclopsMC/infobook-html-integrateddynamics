Blocking Mode

By default, the Crafting Interface will have blocking mode enabled. This will cause the machine to only accept one recipe at a time. It will only start with the next recipe if the previous one has completed.

For machines that can process multiple inputs in parallel, you may disable blocking mode via the settings GUI of the Crafting Interface. When disabling blocking mode, as many inputs as possible will be processed by the machine if you have a multi-amount crafting job.
