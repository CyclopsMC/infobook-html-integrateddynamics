Chemical Crafting

Auto-crafting with chemicals works in exactly the same way as for items, fluids, and energy. You can create recipes within the Logic Programmer (Chemical Recipe), or read all available recipes in a machine using the Machine Reader. These recipes can be inserted in the Crafting Interface for autocrafting in places such as the Crafting Writer or the Storage Terminal.

WARNING: The Chemical Tanks from Mekanism do not allow chemical interaction from all sides by default. So you may have to explicitly allow Input/Output for the connected side within the Chemical Tank GUI.
