Place Items by Tag

Try creating a system using the World Block Exporter that only places wooden logs. Do this using the Tag Values operator to retrieve a list of all items in the logs tag.

演算子
Item Tag Valuesitem_tag_valuesThe Tag values (items) of the given item tag name
  • 文字列
  • リスト
グローバル名: stringItemsByTag(文字列) → リストメソッド名: 文字列.itemsByTag() → リスト
