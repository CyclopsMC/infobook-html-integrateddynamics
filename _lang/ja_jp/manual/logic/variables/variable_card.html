変数カード

変数カードは、このModにおけるロジック転送の主なモードです。変数には、特定の値型を持つ特定の値への参照が含まれます。

変数は静的または動的のいずれかになります。静的変数は一度定義されると、その後変更されることはありません。動的変数は、何らかの形式のロジック評価の結果であり、演算子またはリーダーアスペクトによって実行されます。動的変数の例としては、継続的に変化するマインクラフトのワールドの時間があります。

Dynamic variables will only be evaluated when their value is requested in some way, such as displaying the value in a Display Panel, or being requested as part of the evaluation of another dynamic variable.

各変数カードには一意のIDがあり、ネットワーク内でロジック評価のために内部的に使用されます。ネットワークでエラーが発生すると、デバッグに役立つようにこれらのIDがプレイヤーに表示されます。使いやすさを向上させるために、ラベラーを使用して変数カードにラベルを付けることができます。これにより、プレイヤーはIDではなくラベルを見ることができます。
