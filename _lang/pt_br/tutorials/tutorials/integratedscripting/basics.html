The basics of Scripting

Mendesite blocks are the foundational crafting materials for scripting-related items. Let's create some using the Drying Basin!

Next, we will create some Scripting Disks and Scripting Drives using these Mendesite blocks. Scripting Disks will be able to store scripting data, and Scripting Drives will allow you to insert these disks into your network.
