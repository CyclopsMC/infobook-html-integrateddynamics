Configurar Criação Automática

Neste primeiro tutorial do Integrated Crafting, vamos cobrir a configuração básica da criação automática.

O primeiro requisito para a criação automática é a Interface de Criação, que é uma parte da rede que deve estar voltada para uma máquina, como uma Bancada de Trabalho ou Fornalha. Primeiro, crie tal Interface de Criação e faça-a ficar de frente para uma Bancada de Trabalho.

Em seguida, vamos dizer à Interface de Criação como ela pode criar Tábuas de Carvalho. Faça isso abrindo seu Programador Lógico e escreva um Cartão de Variável que contém uma Receita com como entrada uma Tora de Carvalho e como saída 4 Tábuas de Carvalho (não altere as configurações padrão de correspondência de item). Esta receita DEVE ser criada manualmente, e não preenchida automaticamente usando mods como o JEI.

Finalmente, vamos inserir sua receita recém-criada em sua Interface de Criação para finalizar este primeiro tutorial. Se uma marca de seleção verde for mostrada, isso significa que sua receita é válida e que a receita está agora disponível para a rede para criação automática.
