Filtragem do Terminal de Armazenamento

Uma das muitas características úteis do Terminal de Armazenamento é sua capacidade de filtrar ingredientes. Neste tutorial, exploraremos dois métodos de filtragem diferentes: filtragem baseada em texto e filtragem baseada em variáveis (apenas a segunda lhe dará uma conquista). Este tutorial usará armazenamento baseado em itens, conforme fornecido pelas Interfaces de Itens do Integrated Tunnels.

Para a primeira parte deste tutorial, filtraremos itens digitando uma certa string no campo de busca. Note que você pode usar uma tecla de atalho para selecionar este campo de busca mais rapidamente. E se você tiver o JEI instalado, pode sincronizar seu conteúdo com o campo de busca do JEI. Por padrão, você buscará dentro do nome do ingrediente. Prefixando a string com '@', você pode buscar por mod. Com '#', as dicas de ferramentas podem ser pesquisadas. Finalmente, '$' consulta as tags de item. Para completar a primeira parte deste tutorial, digite '$plank @minecraft' no campo de busca para mostrar todas as tábuas adicionadas pelo Minecraft.

A segunda parte deste tutorial usa filtragem baseada em variáveis usando os slots de Variável no canto inferior direito da interface do Terminal de Armazenamento. Você pode inserir Variáveis que contêm um certo Predicado, conforme configurado no Programador Lógico do Integrated Dynamics, que recebe o tipo de ingrediente como entrada e um Booleano como saída que indica se o ingrediente dado deve ou não ser mostrado. Para este tutorial, vamos tentar filtrar todos os itens que são encantáveis. Para fazer isso, abra um Programador Lógico, vá para o tipo Predicado, procure pelo predicado "Item Encantável" e escreva-o em uma Variável. Depois disso, insira sua Variável em um dos slots de Variável dentro do Terminal de Armazenamento quando a aba de itens estiver selecionada.
