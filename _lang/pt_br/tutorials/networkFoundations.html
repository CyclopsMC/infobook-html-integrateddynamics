Fundações de Rede

O objetivo principal deste mod é construir redes de controle para automação. Isso é feito usando Cabos Lógicos. Tente fazer uma rede com dez desses cabos.

O próximo passo será criar alguns Cartões de Variável e Transformadores de Variável. Cartões de Variável são usados para armazenar referências a valores na rede. Transformadores de Variável são elementos de criação para componentes que podem ser colocados em Cabos Lógicos para ler e escrever esses valores.
