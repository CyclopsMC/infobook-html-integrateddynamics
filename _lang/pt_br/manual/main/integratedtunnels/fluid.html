Fluidos

Mova fluidos pela rede. Importe fluidos para a rede usando Importadores de Fluido para Interfaces de Fluido. Exporte fluidos da rede usando Exportadores de Fluido a partir de Interfaces de Fluido. Filtre dinamicamente quais fluidos podem passar por uma interface usando uma Interface de Fluido com Filtro.
