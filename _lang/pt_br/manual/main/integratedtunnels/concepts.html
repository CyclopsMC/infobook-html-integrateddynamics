Conceitos

O Integrated Tunnels adiciona 4 tipos de partes para os diferentes tipos de transporte, a saber, Importadores, Exportadores e dois tipos de Interfaces.

Importadores (azuis) sempre tentam puxar de um alvo para as Interfaces (brancas) na mesma rede.

Exportadores (laranjas) sempre tentam empurrar para um alvo, a partir das Interfaces (brancas) na mesma rede. Itens, neste caso, sempre serão transferidos da esquerda para a direita.

Você pode adicionar várias interfaces à sua rede e atribuir prioridades para especificar qual deve ser preenchida/esvaziada primeiro.

Se você quiser especificar quais ingredientes podem passar por uma interface, você pode usar uma interface com filtro. Isso pode ser útil se, por exemplo, você quiser travar inventários para conter apenas certos itens.

Além disso, você também pode adicionar múltiplos Importadores e Exportadores, novamente com prioridades relativas.

Usando as variáveis do Integrated Dynamics, você deve especificar quais itens devem ser movidos. Se você quiser manter as coisas simples e apenas mover tudo, simplesmente use um Cartão de Variável em branco.

Note que se você tiver um mod de autocrafting como o Integrated Crafting instalado, então você poderá usar a configuração Criar nos exportadores. Isso fará com que, se um ingrediente não estiver disponível para exportação, ele seja autocriado primeiro.
