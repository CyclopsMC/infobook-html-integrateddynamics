Power Users

Power users tend to dislike clicking too much as it takes too much time. The Storage Terminal offers a couple of hotkeys to speed up the usage of common tasks such as selecting the search field, browsing through tabs, clearing the crafting grid, and balancing crafting grid slots. These hotkeys are listed at the end of this section.

When you have the JEI mod installed, you will be able to exploit JEI's recipe shift-clicking ability to quickly fill the crafting grid. The source of items is defined by the refill source. Furthermore, you can enable synchronization between the JEI and Storage Terminal search fields.

If you are not satisfied with the provided text-based filtering options, you can insert your own Predicates via Variable Cards. These will should accept the ingredient type as input, and a Boolean as ouput.
