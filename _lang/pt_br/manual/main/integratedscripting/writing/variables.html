Constantes e Variáveis

Todos os tipos de valores do Integrated Dynamics aos quais você está acostumado (como Inteiros, Booleanos, Itens, ...) estão disponíveis em JavaScript, e vice-versa.

A maneira mais direta de criar um valor em JavaScript e torná-lo disponível em um valor estático de Cartão de Variável, é definindo um membro variável em JavaScript usando let ou const.

let define variáveis que podem ser reatribuídas posteriormente no script, como contadores. const define variáveis que não podem ser reatribuídas.

Para usar este valor em um Cartão de Variável, você precisa selecionar o nome do membro da variável dentro do editor de texto do Terminal de Scripting e, em seguida, inserir um Cartão de Variável vazio no slot do lado direito. Agora você pode usar este Cartão de Variável como está acostumado, como visualizar seu valor em um Painel de Exibição.

Criar variáveis via JavaScript é útil principalmente se você precisar criar valores complexos que não são fáceis de criar usando o Programador Lógico, como Strings longas com um valor específico. Abaixo, você pode encontrar alguns exemplos de como criar tais valores.

const myBoolean = true;

let myInt = 123 + 456;

let myString = "";
for (let i = 0; i < 50; i++) {
  myString =+ ".";
}

const myList = [ 1, 1, 2, 3, 5 ];
