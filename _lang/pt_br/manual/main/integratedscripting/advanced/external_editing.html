Edição Externa

Se você tiver acesso ao sistema de arquivos no qual seu jogo singleplayer ou multiplayer está sendo executado, poderá acessar os arquivos de script através de qualquer editor externo. Isso permite que você crie scripts usando seu IDE favorito (como Vi, mas definitivamente não o Emacs).

Os arquivos de script estão acessíveis através do diretório world/integratedscripting/ (veja saves/ para mundos singleplayer). Aqui, você encontrará diretórios para todos os Discos de Scripting por seu id. Dentro de cada diretório de Disco de Scripting, você encontrará todos os arquivos de script armazenados nesse disco.

Você pode modificar com segurança esses arquivos de script através de um editor externo, mesmo que o jogo esteja em execução. Após salvar o arquivo externamente, eles serão atualizados automaticamente no jogo, e as variáveis dependentes também serão atualizadas.

Se o seu editor externo suportar isso (IDEs modernos suportam), modificar um arquivo de script através do Terminal de Scripting também fará com que os arquivos sejam atualizados em seu editor externo.
