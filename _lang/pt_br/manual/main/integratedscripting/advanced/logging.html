Logging

Para scripts complexos, pode surgir a necessidade de depurá-los escrevendo temporariamente em arquivos de log externos.

JavaScript permite que você chame funções como console.log() e console.error() para escrever mensagens para os fluxos de saída Standard Out e Standard Error, respectivamente.

Quando chamar qualquer uma dessas funções, mensagens de log serão escritas respectivamente nos arquivos dependentes do script .stdout e .stderr.

Por exemplo, se o seu script estiver localizado em meuarquivo.js, as mensagens de log serão escritas em meuarquivo.js.stdout.

Esses arquivos .stdout e .stderr estão acessíveis através do seu Terminal de Scripting ou através de edição externa.

Para evitar que os arquivos de log fiquem muito grandes, eles são limitados em seu tamanho. Por padrão, eles são limitados a 2096 linhas. Consulte o administrador do seu servidor se precisar de mais linhas.

function myComplexFunc(value) {
  console.log('Hello world!');
  console.log(`I received value ${value}`);
  return value + 42;
}
