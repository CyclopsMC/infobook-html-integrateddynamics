Básico

A Interface de Criação é usada para dizer à rede onde e como as coisas podem ser criadas.

Ao apontar uma Interface de Criação para uma máquina (como uma Bancada de Trabalho ou Fornalha), esta interface será vinculada a esta máquina para lidar com receitas.

Ao abrir a GUI de uma Interface de Criação, você pode inserir Cartões de Variável que contêm Receitas. Isso dirá à interface que a receita fornecida pode ser criada usando a máquina vinculada.

When you've progressed later in the game, you will be able to craft the Attuned Crafting Interface. It does not require you to manually add recipes to it, as it will read and expose all recipes that are available in the target machine. These are the same recipes that are listed in the Machine Reader Some modded machines may not be supported, which will be shown with a red border when placing the Attuned Crafting Interface.
