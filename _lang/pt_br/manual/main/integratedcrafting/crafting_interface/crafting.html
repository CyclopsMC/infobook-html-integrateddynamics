Processo de Criação

Esta seção explicará como o processo de criação é tratado pelas Interfaces de Criação. Esta informação é essencial para tirar o máximo proveito deste mod e para depurar quaisquer problemas que você possa encontrar.

Quando uma tarefa de criação é iniciada (por exemplo, com o Escritor de Criação), uma (ou mais) Interface(s) de Criação será(ão) usada(s) que tenha(m) uma receita correspondente à tarefa fornecida. A Interface de Criação é responsável por inserir todas as entradas da receita na máquina vinculada. Ela fará isso com base no conteúdo da rede de armazenamento, conforme fornecido pelas interfaces do Integrated Tunnels.

Uma Interface de Criação, no entanto, não importará os resultados da criação da máquina vinculada. O jogador é responsável por garantir que os resultados da criação voltem para a rede. A Bancada de Trabalho é uma exceção para isso, apenas uma Interface de Criação é necessária para criar com ela. A Interface de Criação rastreará as alterações na rede de armazenamento e só marcará a tarefa de criação como concluída quando as saídas esperadas estiverem presentes na rede de armazenamento. Isso pode ser alcançado colocando um importador do Integrated Tunnels ativo na máquina, ou deixando a máquina ejetar automaticamente para dentro da Interface de Criação. Tudo (itens, fluidos, energia) que é inserido na Interface de Criação será inserido na rede de armazenamento para o canal configurado.
