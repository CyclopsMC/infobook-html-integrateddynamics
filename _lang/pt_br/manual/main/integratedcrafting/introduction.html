Introdução

Integrated Crafting é uma extensão para o Integrated Dynamics para criação automática dentro das redes do Integrated Dynamics. A criação automática pode ocorrer com qualquer combinação de itens, fluidos e energia. As duas partes principais necessárias para isso são a Interface de Criação e o Escritor de Criação.

A Interface de Criação pode ser colocada ao lado de qualquer tipo de máquina de criação e pode ser usada para fornecer opções de criação para a rede com base em Cartões de Variável contendo Receitas.

O Escritor de Criação pode então ser usado para iniciar automaticamente tarefas de criação com base nas receitas que são expostas por todas as Interfaces de Criação na rede.
