Receita

Uma receita de criação abstrata, que recebe Ingredientes como entrada e produz outro valor de Ingredientes.

As receitas podem ser criadas visualmente usando o Programador Lógico, ou podem ser lidas de máquinas (suportadas) usando o Leitor de Máquina.

Essas receitas podem ser usadas para mods de autocrafting como o Integrated Crafting.

Para encontrar rapidamente receitas específicas entre os Cartões de Variável existentes dentro de uma GUI, você pode segurar shift para exibir temporariamente qual é o item de saída da receita.

Clicar com shift em um slot de entrada dentro do Programador Lógico permite que você configure propriedades detalhadas deste slot de entrada. Por exemplo, permite indicar se o NBT deve ser considerado ou não, se este item pode ser reutilizado em várias receitas (por exemplo, se não será totalmente consumido após o trabalho de criação) e se alternativas baseadas em tags devem ser consideradas para este item.
