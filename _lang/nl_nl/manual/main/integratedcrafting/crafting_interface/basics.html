Grondbeginselen

De vervaardigingsinterface wordt gebruikt om het netwerk te vertellen waar en hoe dingen vervaardigd kunnen worden.

Door een vervaardigingsinterface tegen een machine te laten wijzen (zoals een werkbank of oven), zal deze interface gekoppeld worden met de machine om recepten te verwerken.

Bij het openen van de GUI van een vervaardigingsinterface, kun je variabelekaarten toevoegen die recepten bevatten. Dit vertelt de interface dat het gegeven recept vervaardigd kan worden door de gekoppelde machine te gebruiken.

When you've progressed later in the game, you will be able to craft the Attuned Crafting Interface. It does not require you to manually add recipes to it, as it will read and expose all recipes that are available in the target machine. These are the same recipes that are listed in the Machine Reader Some modded machines may not be supported, which will be shown with a red border when placing the Attuned Crafting Interface.
