Delay

In some situations, you might need to maintain some kind of history of variable values. When you for instance want to take the average or difference over the last values.

For this, you can use a Delay, with which you can maintain the history of the values of a given variable. You can do this by inserting a Variable Card in the top slot for which you want to maintain the history. You can now create now Variable Cards that will refer to a list of values, which corresponds to the history. The referenced (or any other) Variable Card must remain in the top slot for the delay to work.

The oldest values correspond to the last elements in the list. You can choose the maximum length of the history from within the gui.
