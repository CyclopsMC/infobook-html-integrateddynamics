Proxy

A Proxy can be used to make a dynamic reference to a certain Variable Card. It allows you to insert a Variable Card in the top slot, and write a new Variable Card, which will dynamically take the value of the Variable Card that is present in the proxy. Note that the referenced (or any other) Variable Card must remain in the top slot for the proxy to work.

This is useful in situations where you have a chain of Variable Cards, but you still would like to be able to change one value manually. You could then proxy that variable, which will allow it to be swapped with another variable at any time.

Just as with the Materializer, Variable Cards can be piped in and out of it.
