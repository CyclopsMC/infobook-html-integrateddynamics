Введение

Integrated Crafting — это расширение для Integrated Dynamics, которое позволяет автоматизировать крафт в сетях Integrated Dynamics. Автокрафт может происходить с любой комбинацией предметов, жидкостей и энергии. Для этого требуются две основные части: Интерфейс Крафта и Писатель Крафта.

Интерфейс Крафта можно разместить рядом с любым устройством для крафта, и он используется для предоставления сети вариантов крафта на основе Карт Переменных, содержащих Рецепты.

После этого Crafting Writer можно использовать для автоматического запуска заданий по созданию крафта на основе рецептов, представленных всеми Crafting Interfaces в сети.
