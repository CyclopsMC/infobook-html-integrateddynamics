Использование

Терминал Хранилища содержит вкладки со всеми активными сетями хранения в текущей подключенной сети, опционально разделённые по каналам. Например, если у вас подключено несколько Энергетических Батарей, вы увидите сумму всей энергии во вкладке энергии. При установке других модов, добавляющих возможности хранения, таких как предметы и жидкости из Integrated Tunnels, вы также сможете взаимодействовать с ними. Внизу каждой вкладки отображаются общее количество и максимальная ёмкость. В дальнейшем в этой книге мы будем называть вещи, которые могут храниться (например, предметы, жидкости и энергию), ингредиентами.

Помимо просмотра ингредиентов, вы также можете с ними взаимодействовать. Как и с любым инвентарём, это делается путём клика для подбора чего-либо и повторного клика для размещения. Для определённых типов ингредиентов, таких как энергия и жидкости, при клике в вашем инвентаре вам нужно будет выбрать контейнерный предмет, например батареи или резервуары. Помимо кликов, вы также можете использовать Shift-клик для перемещения больших объёмов.

При выборе ингредиента из сети хранения вы можете увеличить его количество на небольшую величину, снова кликнув по слоту сети хранения. Shift-клик увеличит его на большую величину. Аналогично, вы можете уменьшить количество правым кликом. То же самое можно сделать, прокручивая колесико мыши над ингредиентом сети хранения.

При выборе ингредиента из сети хранения вы можете переместить небольшое количество правым кликом, всё, что удерживается, левым кликом, и половину того, что удерживается, средним кликом.

Используя поле поиска, вы можете фильтровать отображаемые ингредиенты по имени. Глубокий поиск среди результатов может быть выполнен путём указания дополнительных параметров, разделённых пробелом. Условия or могут быть определены с помощью '|'.

Через кнопки слева вы можете переключаться между различными вариантами сортировки, такими как упорядочивание по имени и количеству.

Доступны различные префиксы для определения цели поиска. По умолчанию поиск выполняется в имени ингредиента. Добавив префикс '@', вы можете искать по моду. С помощью '#' можно искать в подсказках. Наконец, '$' позволяет выполнять запросы по тегам предметов. Например, условие поиска '$plank @minecraft' покажет все доски, добавленные Minecraft.

Если доступна сеть хранения предметов, вы также получите дополнительную вкладку крафта. Она работает аналогично вкладке предметов, но с интегрированной сеткой крафта. Она имеет несколько полезных встроенных функций, таких как очистка сетки в ваш инвентарь или сеть, балансировка сетки и настройка поведения автозаполнения сетки после крафта.

По умолчанию терминал занимает столько места, сколько возможно. Используя кнопку Размер Сетки, можно изменить это значение, чтобы масштабировать только по высоте, ширине или использовать фиксированное количество строк и столбцов (маленький, средний или большой). Если вы хотите отобразить ещё больше слотов, попробуйте настроить Масштаб GUI в настройках Minecraft. Поведение различных размеров сетки можно настроить с помощью клиентского конфигурационного файла.

Все настройки терминала (кнопки, поля поиска, вкладки и т.д.) сохраняются для каждого игрока и каждого терминала. Чтобы настроить, какие настройки будут загружаться для новых терминалов, нажмите кнопку наковальни в правом нижнем углу интерфейса терминала.
