Смещения

By default, parts will target the direct neighboring position. However, when Part Enhancements are applied, the target position can be changed through the part's Offset screen.

Смещения всегда относительны к целевой позиции части и ограничены количеством примененных Улучшений Части.

Offsets can either be set manually, or they can be set visually using the Wrench. When setting the Wrench to offset-mode, a position can be stored by shift + right-clicking. This position can be saved into parts by right-clicking them with that Wrench.

Смещения могут быть не только статически определены, но и быть §одинамическими§r с использованием Карт Переменных.
