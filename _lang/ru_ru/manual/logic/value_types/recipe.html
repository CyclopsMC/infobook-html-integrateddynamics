Recipe

An abstract crafting recipe, that takes Ingredients as input and outputs another Ingredients value.

Recipes can be created visually using the Logic Programmer, or they can be read from (supported) machines using the Machine Reader.

These recipes can be used for autocrafting mods such as Integrated Crafting.
