Оператор

Значение, содержащее ссылку на Оператор. Операторы могут быть частично применены для более гибкой оценки на основе концепции Каррирования. Это часть концепции метапрограммирования (поищите это, это реально!), где операторы рассматриваются как данные другими операторами. По сути, это позволяет вам создавать новые операторы самостоятельно, которые затем можно использовать для более сложных операций.

Например, вы можете сохранить оператор сложения Оператора на карте переменной. Сделайте это, начав вводить имя оператора, выбрав его из выпадающего списка автозаполнения и нажав на него. Затем вы можете частично применить его с одной Целочисленной Картой Переменной, что приведет к созданию нового виртуального оператора с одним входным и одним выходным типом, который внутренне содержит ссылку на примененное Целое Число. Применение другого Целого Числа даст результат в виде Целого Числа.

Другие операции с Операторами доступны в Программаторе Логики для более сложных вычислений, таких как свертывание списка в одно значение или фильтрация списка на основе Предиката.
