Setup Autocrafting

In this first Integrated Crafting tutorial, we are going to cover the basic setup of autocrafting.

The first requirement for autocrafting is the Crafting Interface, which is a network part that must be faced to a machine, such as a Crafting Table or Furnace. First, craft such a Crafting Interface, and make it face a Crafting Table.

Next, we are going to tell the Crafting Interface how it can craft Oak Planks. Do this by opening up your Logic Programmer, and write a Variable Card that contains a Recipe with as input an Oak Log, and as output 4 Oak Planks (do not change the default item match settings).

Finally, let's insert your newly created recipe into your Crafting Interface to finish this first tutorial. If a green checkmark is shown, this means that your recipe is valid, and that the recipe is now available to the network for autocrafting.
