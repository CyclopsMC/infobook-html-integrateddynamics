Фильтрация Терминала Хранилища

Одной из многих полезных функций Терминала Хранилища является возможность фильтровать ингредиенты. В этом учебнике мы рассмотрим два различных метода фильтрации: текстовую фильтрацию и фильтрацию на основе переменных (только второй метод даст вам достижение). Этот учебник будет использовать хранение предметов, предоставляемое Интерфейсами Предметов из Integrated Tunnels.

В первой части этого учебника мы будем фильтровать предметы, вводя определённую строку в поле поиска. Обратите внимание, что вы можете использовать горячую клавишу для быстрого выбора этого поля поиска. Если у вас установлен JEI, вы можете синхронизировать его содержимое с полем поиска JEI. По умолчанию поиск выполняется в имени ингредиента. Добавив префикс '@', вы сможете искать по моду. С помощью '#' можно искать в подсказках. Наконец, '$' позволяет выполнять запросы по тегам предметов. Чтобы завершить первую часть этого учебника, введите '$plank @minecraft' в поле поиска, чтобы показать все доски, добавленные Minecraft.

Вторая часть этого учебника использует фильтрацию на основе переменных с использованием слотов Переменных в правом нижнем углу графического интерфейса Терминала Хранилища. Вы можете вставить Переменные, содержащие определённый Предикат, настроенный в Программаторе Логики из Integrated Dynamics, который принимает тип ингредиента на входе и возвращает Логическое значение, указывающее, следует ли показывать данный ингредиент. Для этого учебника попробуем отфильтровать все предметы, которые можно зачаровать. Для этого откройте Программатор Логики, перейдите к типу Предикат, найдите предикат "Предмет Зачаровываемый" и запишите его на Переменную. После этого вставьте вашу Переменную в один из слотов Переменных внутри Терминала Хранилища, когда выбрана вкладка предметов.
