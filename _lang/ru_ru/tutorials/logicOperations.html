Логические Операции

Хотя отображение значений, полученных от считывателей, может быть полезным, всё становится действительно интересным, когда вы начинаете комбинировать определённые значения с другими значениями с помощью Операторов. Все Операторы можно найти в Программаторе Логики.

Начните с создания Программатора Логики, который существует как в виде предмета, так и в виде блока. Откройте его графический интерфейс, чтобы увидеть список всех операторов.

Далее найдите оператор Целочисленный и нажмите на него. Это откроет окно, в котором вам нужно ввести число. После того, как вы это сделаете, вставьте пустую Карту Переменной, чтобы связать это Целое Число с Картой Переменной. Когда закончите, повторите это ещё раз для другого Целого Числа.

Наконец, найдите оператор Сложение и откройте его. Это позволит вам сложить два Числа. Вставьте свои две Целочисленные Карты Переменных в эти слоты и свяжите их с новой Картой Переменной.

After writing a value to a Variable Card, you can always modify it again later by inserting it again in the Logic Programmer.

As Integrated Dynamics evaluates variables in a lazy manner, this Addition Variable Card does not contain the addition of these Integers directly, but the addition will only be performed when the value is needed in the network, based on the variable references. For this, the network has to be able to find these referenced Variable Cards. For this, you will need to craft and place a Variable Store. Insert the two Integer Variable Cards into this store to make them available to the network.

Finally, you can now connect a cable to the Variable Store, attach a Display Panel, and insert the Addition Variable Card to display the addition.

The Logic Programmer and Variable Stores are going to be your best friends when you want to do more complex automation.
