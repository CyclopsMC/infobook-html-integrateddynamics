Основы Сетей

Главная цель этого мода — создание управляющих сетей для автоматизации. Это делается с помощью Логических Кабелей. Попробуйте создать сеть из десяти таких кабелей.

Следующим шагом будет создание нескольких Карт Переменных и Трансформаторов Переменных. Карты Переменных используются для хранения ссылок на значения в сети. Трансформаторы Переменных — это элементы для создания компонентов, которые можно разместить на Логических Кабелях для чтения и записи этих значений.
