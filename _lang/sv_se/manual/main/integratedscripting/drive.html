Scripting Drive

Each Scripting Drive is a block that is connected to your Integrated Dynamics network that accepts a single Scripting Disk, which allows the disk's script contents to be exposed to the network.

It is crafted by combining a Variable Store and Jukebox with some Scripting Disks.
