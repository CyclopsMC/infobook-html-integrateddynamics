Koncept

Integrated Tunnels lägger till 3 typer av delar för de olika transporttyperna, nämligen importörer, exportörer och gränssnitt.

Importörer (blå) försöker alltid dra från ett mål, till gränssnitt (vit) i samma nätverk.

Exportörer (orange) försöker alltid trycka till ett mål, från gränssnitt (vit) i samma nätverk. Föremål kommer i detta fall alltid överföras från vänster till höger.

Du kan lägga till flera gränssnitt till ditt nätverk och tilldela prioriteter för att specificera vilka som bör fyllas/tömmas först.

Dessutom kan du även lägga till flera importörer och exportörer, ännu en gång med relativa prioriteter.

Med Integrated Dynamics-variabler måste du specificera vilka föremål som bör flyttas. Om du vill låta det vara enkelt och bara flytta allting, använd bara ett tomt variabelkort.

Observera att om du har installerat en mod för automatisk tillverkning, t.ex. Integrated Crafting, kommer du kunna använda inställningen tillverka i exportörerna. Detta ser till att om en ingrediens inte är tillgänglig för export så bör den tillverkas.

Note that if you have an autocrafting mod such as Integrated Crafting installed, then you will be able to use the Craft setting in exporters. This will make it so that if an ingredient is not available for export, that it should be crafted.
