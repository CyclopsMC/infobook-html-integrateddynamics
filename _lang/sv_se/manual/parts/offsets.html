Offsets

By default, parts will target the direct neighbouring position. However, when Part Enhancements are applied, the target position can be changed through the part's Offset screen.

Offsets are always relative to the part's target position, and are limited by the number of Part Enhancements that have been applied.

Offsets can either be set manually, or they can be set visually using the Wrench. When setting the Wrench to offset-mode, a position can be stored by shift + right-clicking. This position can be saved into parts by right-clicking with that Wrench.

Offsets can not only be statically defined, buy they can also be dynamic using Variable Cards.
