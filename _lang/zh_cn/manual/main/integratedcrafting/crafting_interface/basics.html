Basics

The Crafting Interface is used to tell the network where and how things can be crafted.

By pointing a Crafting Interface to a machine (like a Crafting table or Furnace), this interface will be bound to this machine to handle recipes.

When opening the GUI of a Crafting Interface, you can insert Variable Cards that hold Recipes. This will tell the interface that the given recipe can be crafted using the bound machine.
