Configuration

Like most Integrated Dynamics parts, you can configure things in the Crafting Interface such as the channel, priority and tick rate.

Additionally, you can change the target side separately for each ingredient type. This is useful if the bound machine accepts different ingredient types in different sides. For example, items may be accepted from the sides, while fluids may be accepted from the top.
