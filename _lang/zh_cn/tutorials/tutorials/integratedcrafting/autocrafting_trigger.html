Trigger Autocrafting

In this second Integrated Crafting tutorial, we will start a crafting job for Oak Planks using the Crafting Writer.

First let's start by crafting a Crafting Writer. This will allow you to automatically start crafting jobs for a given recipe, item, fluid or energy. Once you have this, place the writer anywhere on the network that has your Crafting Interface from the previous tutorial.

Before we tell the Crafting Writer to craft Oak Planks, we first need to insert some Oak Logs into the network. In order to do this, attach some storage (like a Chest) to the network using an Integrated Tunnels Item Interface, and make sure that the Chest contains at least one Oak Log.

Now, we tell the Crafting Writer to start crafting Oak Planks. Do this by creating a Variable Card containing Oak Planks using your Logic Programmer, and insert this into the Crafting Writer.

If you configured everything correctly, you will be able to see that one Oak Log has been converted into 4 Oak Planks. Once you remove these planks, 4 new planks will be crafted if other logs were available. You can play around with the Crafting Writer aspect settings to for example ignore the storage contents, and craft as many planks as possible.
